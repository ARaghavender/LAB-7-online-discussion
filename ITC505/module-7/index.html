HTTP Status Codes

Name Numerical Code Purpose Situation
Request Entity Too Large 413 This status code indicates that the request is larger than the server is willing or able to process. If a client attempts to upload a file that exceeds the server's maximum file size limit.
Temporary Redirect 307 This status code indicates that the requested resource resides temporarily under a different URI. When a website is temporarily moved to a new domain or URL, and users should be redirected to this temporary address.
Partial Content 206 This status code is used when the server is delivering only part of the resource due to a range header sent by the client. When a client requests a specific range of bytes from a video file for streaming.
Upgrade Required 426 This status code indicates that the client should switch to a different protocol. If a server wants to force the client to switch from HTTP/1.1 to HTTP/2 for better performance.
Not Extended 510 This status code is used to further extend the request and requires further extensions to be made by the client. When the server requires more extensions to fulfill the request, typically used in conjunction with the 'OPTIONS' request method.
